# !!description

## 소개

기본 인터페이스와 DTO 개수별 기본 동작이 정의된 추상 클래스들의 패키지 입니다.

## 인터페이스 전략

- 기본 메소드는 ID 제네릭을 바탕으로 작동
- 인터페이스의 DTO 별 구현체는 다릅니다.
- 인터페이스에서 작성한 것들은 최소 로직입니다.
- 인터페이스 별 DTO 매핑 메소드가 정해져 있습니다.

## 추상 클래스 전략

- 0 dto, 1 dto, 2 dto.. 간의 관계는 없습니다.
- 개별 메소드들을 별도로 정의하여 사용합니다.
- 모든 추상 메소드에서 사용하는 공동 메소드는 헬퍼 클래스를 이용합니다.
